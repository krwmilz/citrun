C It Run

bb pkg src home

C It Run lets you watch code execute. It attempts to give some sense of what a running program is really doing.

It is intended to help programmers get an intuitive feel for their programs. It associates visual patterns with execution paths, shows pauses in execution, and shows hot execution paths.

C It Run instruments source code using source to source transformation. A wrapper for build systems and an OpenGL viewer are included.
