C It Run

.. pkg src

C It Run is a suite of tools that lets you watch your source code execute. It provides insight into what your program is actually doing.

The C/C++ programming languages are supported on Unix like systems.

Packages

Development packages are being created for the following platforms and architectures:

  • Darwin 15 x86_64: citrun-0_0.darwin_15.x86_64.tbz2
  • Debian 8.4 amd64:
  • OpenBSD 5.9 amd64: citrun-0.tgz
  • OpenBSD 5.9 sparc64: citrun-0.tgz

c citrun run

Self instrumented development packages are being created for the following platforms and architectures:

  • OpenBSD 5.9 sparc64: ccitrunrun-0.tgz

Source

There are two methods available:

  • Distribution archives citrun-0.tar.gz or ccitrunrun-0.tar.gz
  • Read only repository at http://git.0x30.net/citrun

Documentation

The distribution archive comes with these manuals:

  • citrun-gl(1)
  • citrun-inst(1)
  • citrun-wrap(1)

Building

Dependencies

  • C++11 capable compiler
  • DejaVu Sans Mono
  • freeglut and glew
  • FreeType2
  • Jam
  • LLVM and Clang > 3.4
  • OpenGLES >= 2.0

Unpack the distribution archive, cd to the folder created and jam: 

 $ jam

Run the tests with prove: 

 $ prove
