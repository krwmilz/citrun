C It Run

C It Run is a suite of tools that lets you watch your C source code execute. Its main purpose is to provide insight into what your program is actually doing.

Packages

No official release yet. Current development packages are being created by continuous integration:

  • OpenBSD sparc64
  • OpenBSD amd64
  • Debian amd64
  • Mac OS X amd64
