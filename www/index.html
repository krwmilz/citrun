C It Run

C It Run is a suite of tools that lets you watch your C source code execute. Its main purpose is to provide insight into what your program is actually doing.

Currently the C/C++ programming languages are supported on Unix like systems.

Packages

No official release yet. Current development packages are being created by continuous integration:

  • OpenBSD sparc64
  • OpenBSD amd64
  • Debian amd64
  • Mac OS X amd64

Source

There are two methods available:
  • Development tarball or
  • Read-only Git repository: http://git.0x30.net/citrun

Documentation

 $ man citrun_wrap

The manual pages that come with the package should be enough to get you going. If not then they should be improved.
