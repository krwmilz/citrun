C It Run

.. bb pkg src

C It Run is a set of tools to watch source code execute. It supports the C/C++ programming languages on Unix like systems.

Packages

  • Darwin 15 amd64/citrun-0_0.darwin_15.x86_64.tbz2 report (stale)

Building

Dependencies

  • C++11 capable compiler
  • DejaVu Sans Mono
  • freeglut and glew
  • FreeType2
  • Jam
  • LLVM and Clang > 3.4
  • OpenGLES >= 2.0

Unpack the distribution, cd into the created directory and jam: 

 $ jam

Run the tests with prove: 

 $ prove
