C It Run

.. man pkg src

C It Run is a suite of tools that lets you watch your source code execute. It provides insight into what your program is actually doing.

The C/C++ programming languages are supported on Unix like systems.

Packages

No official release yet.

Development packages are being created for the following platforms and architectures:

  • OpenBSD 5.8 sparc64
  • OpenBSD 5.8 amd64
  • Debian 8.4 amd64
  • Darwin x86_64

Self Instrumented Packages

C It Run is able to partly instrument itself, which is fun.

Self instrumented development packages are being created for the following platforms and architectures:

  • OpenBSD 5.8 amd64

Source

There are two methods available:

  • Current tarball or
  • Read-only Git repository at http://git.0x30.net/citrun

Documentation

The source tarball comes with these manuals:

  • citrun-wrap(1)
  • citrun-inst(1)
  • citrun_gl(1)

Building

You'll need jam, a c++11 capable compiler and the development packages installed for:

  • LLVM and Clang > 3.4
  • FreeType2
  • OpenGLES >= 2.0
  • freeglut and glew
  • DejaVu Sans Mono

Once that's done get a copy of the source tree, change directory to the root and jam: 

 $ jam

It wouldn't be stupid to run the integration tests either: 

 $ prove
