C It Run

pkg src ..

C It Run is a suite of tools that lets you watch your source code execute. It provides insight into what your program is actually doing.

The C/C++ programming languages are supported on Unix like systems.

Packages

No official release yet.

Current development packages are being created for the following platforms and architecturs:

  • OpenBSD sparc64
  • OpenBSD amd64
  • Debian amd64
  • Darwin x86_64

Source

There are two methods available:

  • Current tarball or
  • Read-only Git repository at http://git.0x30.net/citrun

Documentation

 $ man citrun_wrap

The manual pages that come with the package should be enough to get you going. If not then they should be improved.

Building

You'll need jam, a c++11 capable compiler and the development packages installed for:

  • LLVM and Clang > 3.4
  • FreeType2
  • OpenGLES >= 2.0
  • freeglut and glew
  • DejaVu Sans Mono

Once that's done get a copy of the source tree, change directory to the root and jam: 

 $ jam

It wouldn't be stupid to run the integration tests either: 

 $ prove
