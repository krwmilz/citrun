NAME

citrun-gl — OpenGL viewer

DESCRIPTION

Multiplexes execution data from any running instrumented programs into an OpenGL window.
Started instrumented programs will try connecting to citrun-gl when it is running. When that succeeds execution data will appear in the viewer window.
citrun-gl uses process group information from connected programs to aggregate similar processes together.

FILES

/tmp/citrun.socket
Listen socket.

EXIT STATUS

The citrun-gl utility exits 0 on success, and >0 if an error occurs.
