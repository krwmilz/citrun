NAME

citrun-term — terminal viewer

DESCRIPTION

Listens for connections from running instrumented programs. Only supports one connection at a time.
citrun-term is controlled by the keys “hjkl”:
“j”
scrolls down
“k”
scrolls up
“h”
previous translation unit
“l”
next translation unit

FILES

/tmp/citrun.socket
Listen socket.

EXIT STATUS

The citrun-term utility exits 0 on success, and >0 if an error occurs.
